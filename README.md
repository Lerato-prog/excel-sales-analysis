# 🚲 Bike Sales Analysis (Excel Dashboard)

## 📌 Project Overview
This project analyzes bike purchase behavior using demographic and commute distance data.

## 🛠 Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- Data Cleaning

## 📊 Key Analysis Areas
- Purchase behavior by Gender
- Purchase trends by Age Bracket
- Commute Distance impact on purchases
- Regional and Education breakdown

## 📈 Key Insights
- Middle-aged customers show higher purchase rates.
- Customers commuting 0–1 miles are more likely to purchase.
- Regional differences impact buying patterns.
